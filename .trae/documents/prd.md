## 1. 产品概述

微博热搜异动监测 App 是一款面向品牌市场负责人的移动端舆情快览工具，通过实时捕捉微博热搜榜单中的异常波动，帮助用户快速判断活动投放时机与代言人风险，无需专业舆情团队或监控大屏即可掌握关键变化。

- 核心目标：将热搜异动从"被动获知"变为"主动预警"，让市场负责人在碎片时间内完成品牌舆情判断
- 目标用户：品牌市场总监、市场经理、公关负责人等非专业舆情监测人员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 品牌市场负责人 | 企业邮箱注册 | 查看异动、收藏、发起讨论、查看周报 |
| 协作同事（市场/公关/法务） | 企业邮箱注册 | 接收讨论通知、参与讨论、查看相关话题 |

### 2.2 功能模块

1. **首页**：热搜异动卡片流、四视角切换（我方品牌/竞品动态/代言人/行业议题）、异动摘要与通俗解读
2. **详情页**：异动原因深度解读、排名曲线图、话题截图、收藏操作、发起内部讨论
3. **周报页**：一周热搜机会与风险按影响程度排序、趋势总结、关键事件时间线

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 视角切换栏 | 四个Tab切换：我方品牌、竞品动态、代言人、行业议题，切换后卡片列表实时刷新 |
| 首页 | 异动卡片列表 | 每张卡片展示：热搜话题标题、异动标签（上升/下降/反复/新入）、热度值、通俗语言异动原因（如"竞品新品话题连续两次回升""代言人关联争议词进入榜单"）、时间戳 |
| 首页 | 筛选与排序 | 按时间/热度/风险等级排序，支持仅看风险/仅看机会筛选 |
| 详情页 | 异动概览 | 话题标题、当前排名、历史最高排名、在榜时长、异动类型标签 |
| 详情页 | 排名曲线 | 折线图展示话题排名变化趋势，标注异动节点 |
| 详情页 | 话题截图 | 自动截取微博热搜榜画面快照 |
| 详情页 | 异动原因解读 | 用通俗语言详细解释为什么被判定为异动，包含关联词分析 |
| 详情页 | 收藏操作 | 收藏为观察对象，收藏后可在个人中心查看持续追踪 |
| 详情页 | 发起讨论 | 选择市场/公关/法务等同事，自动带上话题截图、排名曲线和初步判断，发起内部讨论 |
| 周报页 | 周报摘要 | 本周热搜机会数与风险数总览，与上周对比 |
| 周报页 | 机会排序 | 按影响程度排序的热搜机会列表，标注建议行动 |
| 周报页 | 风险排序 | 按影响程度排序的热搜风险列表，标注预警等级 |
| 周报页 | 关键事件时间线 | 一周内关键热搜异动事件的时间轴展示 |

## 3. 核心流程

用户打开 App 后，首页默认展示"我方品牌"视角的热搜异动卡片流。用户可通过顶部 Tab 切换视角查看不同维度的异动信息。点击任意卡片进入详情页，查看排名曲线、话题截图和异动原因深度解读。用户可收藏话题持续观察，或发起内部讨论邀请同事协作。每周可切换到周报页查看一周汇总。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃夜空蓝（#0F172A）为底色，搭配醒目珊瑚红（#FF6B6B）作为风险警示色，翡翠绿（#10B981）作为机会标识色
- **辅助色**：琥珀黄（#F59E0B）用于"关注"状态，冰蓝（#38BDF8）用于信息提示
- **按钮风格**：圆角胶囊按钮，主要操作使用实心填充，次要操作使用描边样式
- **字体**：标题使用思源黑体 Bold，正文使用系统默认无衬线体，数字使用等宽风格
- **布局风格**：卡片式布局，底部 Tab 导航，顶部视角切换栏
- **图标风格**：线性图标，1.5px 描边，圆角端点

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 视角切换栏 | 水平滚动Tab，当前选中项底部高亮条，滑动切换动画 |
| 首页 | 异动卡片 | 圆角卡片，左侧异动类型色条，右上角时间戳，底部通俗语言摘要，点击水波纹效果 |
| 首页 | 筛选栏 | 右上角筛选图标，下拉面板含排序和筛选选项 |
| 详情页 | 异动概览 | 顶部大字标题，排名数字突出显示，异动标签胶囊 |
| 详情页 | 排名曲线 | 面积图折线，异动节点圆点标注，触摸显示详情浮窗 |
| 详情页 | 操作栏 | 底部固定操作栏，收藏按钮（心形图标）和发起讨论按钮（对话框图标） |
| 详情页 | 讨论发起 | 底部滑出面板，同事选择列表，预览带上的截图和曲线 |
| 周报页 | 周报摘要 | 顶部双数字卡片（机会/风险数量），对比箭头指示涨跌 |
| 周报页 | 排序列表 | 分组列表，机会组绿色左边框，风险组红色左边框，每项显示影响星级 |
| 周报页 | 时间线 | 垂直时间线，节点圆点+时间+事件摘要，左右交替排列 |

### 4.3 响应式设计

- 移动优先设计，目标屏幕宽度 375px-428px
- 底部导航栏适配安全区域（safe area）
- 卡片列表支持下拉刷新和无限滚动加载
- 触摸优化：卡片点击热区 ≥ 44px，滑动切换视角

### 4.4 3D 场景指引

不适用，本项目无 3D 场景需求。
